"Subfolder" is the folder in the AO planetmap directory where your map will be stored.
Bin Structure: 
  Multi:		One binfile per worker
  Single:		One binfile for entire map

Out folder:		Which folder to export the map to when it has been compiled?
Slice Padding:	Tells the compiler to pre-allocate the specified amount of KiB for each slice,
				ensuring slices file positions will never change between releases.
