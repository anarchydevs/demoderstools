This is a list of the maps choices people will see ingame.
Rightclick the list and hit:
  Add    to add a new version.
  Edit   to edit an existing version.
  Remove to remove an existing version.

  Name & Coordsfile accept the following variables:
    {name} will be replaced with maps name (provided in the Map Information tab).
    {binname} will be replaced with bin files name (provided in the Map Information tab).
    {version} will be replaced with maps version (provided in the Map Information tab).
    {subfolder} will be replaced with the maps Sub directory (provided in the Map Information tab).
